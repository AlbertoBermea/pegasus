# Pegasus Release Notes - Summary

## Versioning
Name reflects ```major```, ```minor``` and ```patch``` concepts:
```
	major.minor.patch
```

Breaking changes may occur on ```mayor``` and ```minor``` numbers. ```Patches```, are not expected to affect compatibility.
